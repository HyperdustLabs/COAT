# 01 — Simple chat agent

Goal: smallest possible end-to-end run of the runtime.

* In-process runtime, memory ConcernStore, memory DCNStore, stub LLM
* One user input → one extracted concern → one advice → one injection
* No persistence, no daemon, no network

This example becomes runnable at **M1**. See
[`docs/07-mvp/milestones.md`](../../docs/07-mvp/milestones.md).
