# Non-goals

The MVP intentionally postpones:

- Complex neural-style DCN learning (v0.3+)
- Fully automatic Meta Concern generation (v0.3+)
- Token-level rewriting tools (v0.3+)
- Federated DCN across multiple agents (v0.4+)
- Real-time risk closure / autonomous tool execution (v0.3+)
- Large-scale long-term self-evolution

These are explicit non-goals — please don't reintroduce them under
different names.
